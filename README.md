### EQD2 Clinical Re-irradiation Calculator
Web-based radiobiology tool for re-irradiation planning

**Features:**
- Scheme A vs B comparison (BED & EQD2 for α/β=10 and α/β=3)
- OAR dose tracker with Pass/Warning/Exceeds tolerance badges
- Tissue tolerance estimator with time-repair discount
- Patient case header + PDF export for clinical documentation
